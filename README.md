A powerful WordPress plugin that enables automatic user login via custom URL slugs. Perfect for creating secure, shareable access links for clients, team members, or specific user groups.

- **Clean Direct URLs**: Access links work directly at root level (e.g., `yoursite.com/my-access-link`)
- **No URL Prefix**: No ugly `/qaw/` or other prefixes in your URLs
- **Smart Conflict Detection**: Automatically prevents conflicts with existing pages, posts, and WordPress content
- **User Mapping**: Assign each slug to a specific WordPress user
- **Usage Limits**: Set maximum number of uses per slug
- **Expiration Dates**: Auto-expire slugs after a specified date/time
- **Custom Redirects**: Redirect users to any page after login
- **Rate Limiting**: Protect against brute-force attacks
- **Access Logging**: Track all
